# 🏦 ATM Banking System using Java

## 📌 Description
The ATM Banking System is a Java desktop application that simulates the basic functions of an ATM machine. Users can create an account and log in securely using a card number and PIN. After logging in, users can deposit money, withdraw cash, check their account balance, view a mini statement, and change their PIN. The project is built using Java Swing for the graphical user interface and JDBC with MySQL for storing user and transaction data.

## ✨ Features
- Account Registration
- Secure Login (Card Number & PIN)
- Deposit Money
- Withdraw Money
- Balance Enquiry
- Mini Statement
- PIN Change

## 🛠 Technologies Used
- Java
- Java Swing
- JDBC
- MySQL

## ⚙️ How to Run
1. Clone this repository
2. Open the project in VS Code / IntelliJ / Eclipse
3. Setup MySQL database
4. Run `Login.java`
